Getting Started

Install Kabanero Foundation

Installing Kabanero Foundation will allow you to acitvate, manage, and share collections with your team all while maintaining governance over each piece of the collection. Kabanero Foundation is installed in a Kubernetes cluster using a Kubernetes Operator. Today Kabanero uses Origin Community Distribution of Kubernetes (OKD) version 3.11, but in the future we plan on expanding to use upstream Kubernetes and Red Hat OpenShift.

The full suite of Kabanero Foundation components that will be installed include: Istio, Knative, Eventing, Serving, and Tekton pipelines. Follow the instructions to set up Kubernetes, install the Kabanero Operator, and set up your Tekton pipeline.

Prerequisites: Kubernetes and OKD Version 3.11
